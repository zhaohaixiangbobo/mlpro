# UI 优化计划：参数单行显示与表格滚动

针对您的需求，我制定了以下修改计划，旨在优化表格的展示效果，使其更整洁且适应不同屏幕宽度。

## 1. 主要参数单行显示
**目标**：将分类、回归、聚类三个表格中的“主要参数”列改为单行排列，不再换行占用垂直空间。
**实施**：
- 修改 `Dashboard.tsx` 中三个表格（分类、回归、聚类）的 `params` 列渲染逻辑。
- 将参数容器的样式从 `flex-wrap: wrap`（自动换行）改为 `flex-wrap: nowrap`（不换行）。
- 移除或调整 `maxWidth` 限制，允许列宽根据内容自然撑开（配合表格滚动）。

## 2. 回归指标对比表格滚动支持
**目标**：解决表格内容（特别是加入长参数列后）超出 Card 边界的问题，通过横向滚动条展示完整内容。
**实施**：
- 在 `renderRegressionSection` 的 `Table` 组件中添加 `scroll={{ x: 'max-content' }}` 属性。
- 这将启用 Ant Design 表格的横向滚动功能，确保表格始终在 Card 容器内，超出部分可通过滑动查看。
- **同步优化**：为了保持一致性，也会给 **分类** 和 **聚类** 的表格添加相同的滚动属性。

## 总结
通过以上修改，所有算法的参数将整齐地排列在一行中，且表格不再会撑破布局，您可以通过左右滑动来查看完整的指标和参数信息。
