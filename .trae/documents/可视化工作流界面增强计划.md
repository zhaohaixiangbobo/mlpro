# Add Download Functionality to File List

## 1. Backend Changes
### `d:\2-code\mlpro\backend\app\api\data.py`
- Import `FileResponse` from `fastapi.responses`.
- Add a new API endpoint `GET /download/{filename}`:
    - Validate file existence.
    - Return the file using `FileResponse` with `media_type='application/octet-stream'` and `filename` header.

## 2. Frontend Changes
### `d:\2-code\mlpro\frontend\src\services\api.ts`
- Add `downloadFile` function:
    - Make a GET request to `/data/download/{filename}` with `responseType: 'blob'`.
    - Create a temporary `<a>` element to trigger the browser download using `window.URL.createObjectURL`.

### `d:\2-code\mlpro\frontend\src\pages\DataImport.tsx`
- Import `DownloadOutlined` icon.
- Add `handleDownload` function to call the service.
- Add a **Download Button** next to the Delete button in the file list item.
