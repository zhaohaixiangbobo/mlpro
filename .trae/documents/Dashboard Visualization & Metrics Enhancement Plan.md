# 可视化结果与指标展示优化方案

## 1. 回归分析优化 (MAE 图表与表格)
### MAE 误差对比图表
- **问题**：当前图表使用 `autoFit` 导致算法较少时柱子过宽，且分布过于分散，不够美观。
- **解决方案**：
    - **调整柱宽**：设置更严格的 `maxColumnWidth` (如 30px) 和 `minColumnWidth`，防止柱子过宽或过窄。
    - **优化布局**：调整 X 轴的 `scale` 或 `padding` 配置，使柱子在数量较少时也能保持合理的间距，而不是撑满整个容器。
    - **保持配色**：保留“最优值(绿色) vs 其他(蓝色)”的高亮逻辑。

### 回归指标对比表格
- **问题**：缺少算法参数列，且未标注各项指标的最优值。
- **解决方案**：
    - **新增“主要参数”列**：显示算法的超参数（如 `n_estimators`, `learning_rate` 等），保持与其他表格风格一致。
    - **高亮最优值**：
        - **R2 Score**：高亮**最大值**（红色加粗）。
        - **MAE**：高亮**最小值**（红色加粗）。
        - **RMSE**：高亮**最小值**（红色加粗）。

## 2. 分类指标详情优化
- **问题**：“主要参数”显示不全（目前仅显示前 3 个）。
- **解决方案**：
    - **展示所有参数**：移除参数列渲染中的 `.slice(0, 3)` 限制。
    - **样式调整**：如果参数过多，采用自动换行展示，确保所有参数可见。

## 3. 聚类评估指标优化
- **问题**：表格中缺少相关参数信息。
- **解决方案**：
    - **新增“主要参数”列**：在聚类评估表格中增加参数列，展示如 K-Means 的 `n_clusters` 或 DBSCAN 的 `eps` 等关键参数。
    - **统一风格**：使用 Tag 组件展示参数键值对。
